# Analisis_Demografico_De_La_Guajira_Riohacha_2005-2018
Análisis demográfico del departamento de La Guajira y el municipio de Riohacha usando datos del DANE. Se estudian estructura poblacional, pobreza, migración, mortalidad y brechas urbano-rurales, destacando retos sociales, económicos y de salud pública en la región.
